# Implementation Plan: scheduled_hours_per_week + Part-Time Eligibility Fix

**Branch**: `093-part-time-scheduled-hours` | **Date**: 2026-06-10 | **Spec**: [spec.md](spec.md)
**GitHub Issue**: #282

## Summary

Add `scheduled_hours_per_week` as an optional census field that flows from the census staging model through eligibility calculations, replacing the hardcoded `2080` annual hours assumption with `COALESCE(scheduled_hours_per_week, 40.0) * 52.0`. Add a `part_time_new_hire_pct` config param so simulation-generated hires can also be part-time. Add a "Match Census" magic button endpoint that reads the census and auto-populates the config.

### Phase 2: Eligibility Hours Formula Fix

**Goal**: Both eligibility models use actual scheduled hours instead of hardcoded 2080.

**2.1 — `int_employer_eligibility.sql`**

Locate all occurrences of `2080.0` and `2080` in the hours calculation block (lines 271-300). Replace:

```sql
-- BEFORE
days * (2080.0 / 365.0)
-- full year: 2080

-- AFTER
days * (COALESCE(ae.scheduled_hours_per_week, 40.0) * 52.0 / 365.0)
-- full year: COALESCE(ae.scheduled_hours_per_week, 40.0) * 52.0
```

Alias `ae` is the census/staging join alias — verify at read time.

**2.2 — `int_eligibility_computation_period.sql`**

Same substitution for IECP year1_hours, year2_hours, plan_year_hours formulas. The `2080.0` constant appears in lines ~189-255.

### Phase 3: New Hire Part-Time Assignment

**Goal**: Simulation-generated hires can be assigned part-time schedule.

**3.1 — `workforce.py`: Add config param**
```python
class WorkforceSettings(BaseModel):
    total_termination_rate: float = Field(default=0.12, ge=0, le=1)
    new_hire_termination_rate: float = Field(default=0.25, ge=0, le=1)
    part_time_new_hire_pct: float = Field(default=0.0, ge=0, le=1,
        description="Fraction of new simulation hires assigned 20 hrs/week schedule")
```

**3.2 — `int_hiring_events.sql`: Deterministic part-time assignment**

After the `hire_with_age` CTE, add:
```sql
hire_with_part_time AS (
  SELECT
    hwa.*,
    CASE
      WHEN ABS(MOD(
        HASH(CONCAT(CAST(hwa.hire_sequence_num AS VARCHAR), '_pt_', CAST({{ simulation_year }} AS VARCHAR)))::DOUBLE,
        1000000.0)) / 1000000.0
        < {{ var('part_time_new_hire_pct', 0.0) }}
      THEN 20.0
      ELSE NULL
    END AS scheduled_hours_per_week
  FROM hire_with_age hwa
)
```

Include `scheduled_hours_per_week` in the final SELECT.

**3.3 — `int_workforce_previous_year.sql`: Carry forward**

Add to the `previous_year_snapshot` CTE SELECT:
```sql

```

**3.4 — `fct_workforce_snapshot.sql`: Expose in output**

Add `scheduled_hours_per_week` to the final SELECT (near `annual_hours_worked`).

### Phase 4: API Endpoint + Service

**Goal**: `/analyze-part-time-pct` endpoint for magic button.

**4.1 — `files.py`: New Pydantic response model**
```python
class PartTimePctResponse(BaseModel):
    column_present: bool
    headcount: int
    part_time_count: int
    part_time_pct: float
```

**4.2 — File service: `analyze_part_time_pct()` method**
```python
def analyze_part_time_pct(self, workspace_id: str, file_path: str) -> PartTimePctResponse:
    df = pd.read_parquet(self._resolve_census_path(workspace_id, file_path))
    if "scheduled_hours_per_week" not in df.columns:
        return PartTimePctResponse(column_present=False, headcount=len(df),
                                   part_time_count=0, part_time_pct=0.0)
    headcount = len(df)
    part_time = df[df["scheduled_hours_per_week"] * 52 < 1000]
    count = len(part_time)
    return PartTimePctResponse(column_present=True, headcount=headcount,
                               part_time_count=count,
                               part_time_pct=round(count / headcount, 4) if headcount else 0.0)
```

**4.3 — `routers/files.py`: New endpoint**
```python
@router.post("/{workspace_id}/analyze-part-time-pct")
async def analyze_part_time_pct(
    workspace_id: str,
    request: FileValidationRequest,
) -> PartTimePctResponse:
    service = get_file_service()
    try:
        return service.analyze_part_time_pct(workspace_id, request.file_path)
    except ValueError as e:
        raise HTTPException(status_code=422, detail=str(e))
    except Exception as e:
        logger.error(f"Failed to analyze part-time pct: {e}")
        raise HTTPException(status_code=500, detail="Internal server error")
```
